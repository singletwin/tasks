Как пользоваться.

1. Сперва необходимо запустить файл install.php.
  В нем необходимо будет ввести данные для использования mysql.

Будет создана база данных, и таблица в ней(с одной тестовой записью). А также файл с данными для доступа к mysql


2. затем для использования приложения необходимо загружать страницу index.php

имя пользователя и пароль для доступа к приложению одинаковые: valid



3. Если нужна кнопка удаления: раскомментировать в файле edit.php строки 145-154, а в файле view.php строки ачинающиеся с	<!--- <button     ...     и заканчивающиеся  </button> -->  (231-233, 326-328)


4. Время в универсальном международном формате . UTC + 0


5. sql-код создания таблицы (+БД) с одной тестовой записью в файле sql-code.txt . (но рекомендуется стартовать с файла istall.php, который помимо создания базы, таблицы и записи , создает файл конфигурации для последующего обращения к mysql)

6. Валидация данных заполненной формы сделана не очень строгой.. Её хватит для предотвращения вредоносного или уж совсем некорректного ввода  . Но на соответствие реально существующим названиям городов стран и тд данные  не проверяются ...
